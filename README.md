## Yarn CLI

### Install dependencies
```
$ nvm use
...
$ yarn install
```

### Build
```bash
$ yarn build
```
### Run

```bash
// for api-server
$ yarn start:server
```

### Workspaces info

```
$ yarn workspaces info
```
